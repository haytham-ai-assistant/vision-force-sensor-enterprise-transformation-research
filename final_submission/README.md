## 格式调整要点

### 已调整的格式元素
1. **封面页格式**：
   - 论文题目：一号黑体，居中（不超过20字）
   - 作者信息：三号仿宋字
   - 完成日期：三号宋体汉字（二〇二六年二月）

2. **中英文摘要格式**：
   - 中文摘要标题：黑体三号，居中
   - 中文摘要内容：小四宋体，固定行距20磅
   - 英文摘要标题：Arial三号，居中
   - 英文摘要内容：小四Times New Roman

3. **章标题格式**：
   - 使用中文数字：第一章、第二章...
   - 黑体三号，居中
   - 段前24磅，段后18磅

4. **节标题格式**：
   - 一级节标题：阿拉伯数字（1.1），黑体四号，居左
   - 二级节标题：阿拉伯数字（1.1.1），黑体13pt，居左

5. **正文格式**：
   - 中文字体：小四宋体
   - 英文字体：Times New Roman
   - 行距：固定20磅
   - 首行缩进：2个汉字符

6. **页面设置**：
   - 纸张：A4（21.0 cm×29.7cm）
   - 页边距：上3.0cm，下2.5cm，左2.6cm，右2.6cm
   - 页眉：奇数页为章节名，偶数页为"北京大学硕士学位论文"
